# Simple Task Management System

A beginner-friendly full-stack web application for managing personal tasks with a Kanban-style board. Built with React, Node.js, Express, MongoDB, and Prisma. Implements JWT authentication, the Factory design pattern, and SOLID principles throughout.

## Features
- User Authentication (Signup/Login/Logout) via JWT
- Full CRUD for tasks (Create, Read, Update, Delete)
- Task Status Management (Todo, In Progress, Completed)
- Task priority levels: LOW, MEDIUM, HIGH
- Optional due date per task
- Factory Design Pattern for task creation
- Task ownership enforcement (users only see their own tasks)
- Centralized error handling
- SOLID principles followed for clean architecture

## Tech Stack
- **Frontend:** React (Vite), Axios, React Context API
- **Backend:** Node.js, Express
- **Database:** MongoDB with Prisma ORM
- **Auth:** JWT (jsonwebtoken), bcryptjs)
- **Security:** Helmet.js, CORS
- **Styling:** Vanilla CSS
